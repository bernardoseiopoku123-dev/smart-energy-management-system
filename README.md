# smart-energy-management-system


## Overview


The system manages different smart home devices such as:
- Smart Lights
- Security Cameras
- Temperature Sensors

The project demonstrates important OOP concepts including:
- Classes and Objects
- Constructors
- Inheritance
- Encapsulation
- Getters and Setters
- Method Overriding
- Use of super()

## Features

The system allows users to:

1. Display device information
2. Turn devices ON
3. Turn devices OFF
4. Read temperature values
5. Adjust smart light brightness
6. Start camera recording
